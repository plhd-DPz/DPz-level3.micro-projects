# Micro Projects

This repository contains very small functional applications built to bridge the gap between isolated problem-solving and structured software development.

## Purpose

- Move beyond single-problem solutions  
- Practice organizing multi-function programs  
- Learn basic program structure  
- Write self-contained functional tools  

## What qualifies as Level 3

- Functional CLI applications  
- Usually single-file or minimal structure  
- Very limited features  
- Focused on solving one clear problem  

## Example Ideas

- Simple calculator CLI  
- Basic to-do manager  
- File renamer tool  
- Unit converter  
- Random password generator  

## Structure

    app_name/
        main_file
        readme_name.md

## Philosophy

From solving problems to building tools.  
Small scope, complete execution.
